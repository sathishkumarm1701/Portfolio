================================================================================
                              HOW TO USE
================================================================================

VIEW THE BLOG:
1. Home Page: Scroll to "Latest Articles" section
2. Dedicated Page: Visit /blog
3. Individual Post: Click "Read Article" on any card

ADD NEW ARTICLE:
1. Open lib/blogs.ts
2. Add to blogPosts array with required fields
3. Save and deploy

EDIT ARTICLE:
1. Open lib/blogs.ts
2. Find article by ID
3. Edit content, date, tags, etc.
4. Save and deploy

MANAGE CATEGORIES:
- Add new category by creating article with new category name
- Filter by category on blog page
- View all categories in category buttons
