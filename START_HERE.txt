QUICK START (5 MINUTES)
================================================================================

1. Install dependencies:
   npm install

2. Set up environment:
   cp .env.local.example .env.local
   # Edit .env.local with your credentials

3. Run locally:
   npm run dev
   # Visit http://localhost:3000

4. Customize:
   # Edit components/sections/ files

5. Deploy:
   npm i -g vercel
   vercel
