## 🚀 How to Add More Certificates

### Quick Steps

1. **Open the file:**
   ```
   components/sections/Certifications.tsx
   ```

2. **Find the certifications array:**
   ```typescript
   const certifications = [
     // Your certificates here
   ];
   ```

3. **Add your certificate:**
   ```typescript
   {
     title: 'Your Certificate Title',
     issuer: 'Issuer Name',
     date: 'Completed YYYY',
     credentialId: 'Your ID',
     credentialUrl: 'https://link-to-certificate.com',
     description: 'What you learned',
     skills: ['Skill 1', 'Skill 2', 'Skill 3'],
     icon: '🎓',
   }
   ```

4. **Deploy:**
   ```bash
   git add .
   git commit -m "Add new certification"
   git push origin main
   ```
